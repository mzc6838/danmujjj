# 弹幕🐓🐓🐓
> 用来看b站直播时的弹幕用的

## 介绍
使用websocket，没有那种louB的反复http请求奥，就这

## 目前进度：
- 支持查看弹幕消息和送礼物消息

## 画饼：
- 打算配合图形界面，实现美观实用功能（画了一半了 奥利给）
- 整合点歌鸡功能，可以查看目前正在播放的音乐（外部播放器），也可以根据弹幕内容实现歌曲点播
- 有能力的话，实现在图形界面中发送弹幕到直播间，而不用单独打开直播间回复弹幕消息（针对哑巴主播比如我）
